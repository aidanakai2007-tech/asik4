Graph Algorithms in Java

This project implements a weighted graph using vertices with adjacency maps instead of a separate Edge class.

Implemented Classes

- `Vertex`
- `WeightedGraph`
- `Search`
- `BreadthFirstSearch`
- `DijkstraSearch`
- `Main`

Features

- Breadth-First Search (BFS)
- Dijkstra shortest path algorithm
- Generic graph implementation using Java Generics
- Weighted connections stored inside vertices

Example

The `Main` class demonstrates:
- creating vertices
- adding edges with weights
- running BFS
- running Dijkstra algorithm

Technologies

- Java
- Collections Framework (`HashMap`, `HashSet`, `PriorityQueue`, `Queue`)
